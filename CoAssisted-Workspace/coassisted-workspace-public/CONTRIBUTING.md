# Contributing

Thanks for your interest! Please open an issue first for non-trivial changes.

## Setup
1. Fork + clone
2. `./install.sh` to bootstrap the venv
3. `make test` to run the suite (600+ tests)

## Code style
- Black-compatible formatting
- Type hints required on new public functions
- New features need tests
